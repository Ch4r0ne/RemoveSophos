# Uninstall_Sophos
Uninstall Sophos Endpoint Sophos Endpoint Security and Control Version 10.8

Guide:
Um den SEDService (Sophos Endpoint Defense Service) zu deinstallieren benötigst du "SEDuninstall.exe" diese findest du in der Install_Sophos.exe entpacke diese mit einen Zip-Programm und suche nach der "SEDuninstall.exe". Optional kannst du auch "AVRemove.exe" hier finden und zu den Pfad wo sich die Uninstall_Sophos.ps1 befinden hinzufügen.
